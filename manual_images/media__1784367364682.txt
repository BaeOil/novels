# PROJECT CONTEXT — Interactive Choice-Based Novel Platform

## Project Name

เว็บแพลตฟอร์มนิยายออนไลน์แบบเลือกเส้นทางเนื้อเรื่อง
(Interactive Choice-Based Novel Platform)

---

# Project Overview

ระบบเป็น Web Platform สำหรับ “เขียน” และ “อ่าน” นิยายแบบ Interactive Choice-Based Story

ผู้ใช้งานสามารถ:

* อ่านนิยายแบบเลือกตัวเลือก
* เปลี่ยนเส้นทางเนื้อเรื่องได้
* ปลดล็อกตอนจบหลายแบบ
* บันทึกเส้นทางที่เคยเล่น
* ย้อนกลับไปเลือกใหม่ได้

ฝั่งนักเขียนสามารถ:

* สร้างนิยาย
* สร้าง chapter / scene
* เชื่อม scene ด้วย choices
* ดู Story Tree / Flow Diagram
* จัดการเส้นทางของเรื่อง

ระบบจะมีลักษณะคล้าย:

* Visual Novel
* Interactive Fiction
* Choice-Based Narrative Game
* Twine + Web Novel Platform

# Core Concept

ระบบใช้โครงสร้าง:

Novel
→ Chapters
→ Scenes
→ Choices

โดย:

* Scene = หน่วยเนื้อหาหลัก
* Choice = ตัวเลือกเชื่อม scene
* ผู้เล่นจะอ่านทีละ scene
* แล้วเลือก choices เพื่อไป scene ถัดไป

รองรับ:

* branching story
* multiple endings
* hidden routes
* replay system

### user_choice_history

### user_scene_history

# APIs Overview

ประมาณ 65 APIs

## Novel APIs

```http
GET    /api/v1/novels
GET    /api/v1/novels/:id
GET    /api/v1/novels/:id/chapters
POST   /api/v1/novels
PUT    /api/v1/novels/:id
DELETE /api/v1/novels/:id
```

## Chapter APIs

```http
GET    /api/v1/chapters/:id/scenes
POST   /api/v1/chapters
```

## Scene APIs

```http
GET    /api/v1/scenes/:id
GET    /api/v1/scenes/:id/choices
POST   /api/v1/scenes
```

## Choice APIs

```http
POST /api/v1/choices
```

## Authentication APIs

```http
POST /api/v1/auth/login
POST /api/v1/auth/register
GET  /api/v1/auth/profile
```

## Comment APIs

```http
GET  /api/v1/comments
POST /api/v1/comments
```

## Writer APIs

```http
GET /api/v1/writer/dashboard
```
